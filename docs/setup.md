# 環境構築

このドキュメントでは、PFLIGHT2.0 の開発・動作確認で使用している基本環境をまとめます。
開発には STM32CubeIDE を使用し、対象マイコンは STM32F732RET6 です。
機体側では 6 軸 IMU の ICM-42688-P をセンサーとして使用し、サーボおよびモーターの PWM 出力は 50 Hz を前提としています。

## 開発環境

| 項目 | 内容 | 備考 |
| --- | --- | --- |
| IDE | STM32CubeIDE 1.19.0 | STM32 向けの統合開発環境 |
| 対象 MCU | STM32F732RET6 | ファームウェアのビルド・書き込み対象 |

## 使用デバイス

| 種別 | 型番・仕様 | 用途 |
| --- | --- | --- |
| IMU | ICM-42688-P | 6 軸 IMU として加速度・角速度を取得 |
| サーボ / モーター | PWM 50 Hz | サーボおよびモーター制御用の PWM 出力 |
